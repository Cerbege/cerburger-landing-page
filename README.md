# 🍔 CerBurger - Responsive Landing Page

CerBurger is a modern, user-friendly, and fully responsive burger restaurant landing page designed with professional web development principles.

## 🚀 Features
- **Fully Responsive Design:** Seamless experience across desktop, tablet, and mobile devices.
- **Hero Slider:** A smooth-transitioning visual area built using CSS `@keyframes` animations.
- **Modern UI:** Sleek navigation menu, interactive buttons, and a professional color palette.
- **Social Media Integration:** Elegant social media icons to boost brand visibility.

## 🛠️ Tech Stack
- **HTML5:** Semantic structure and layout.
- **CSS3:** Flexbox layout, custom animations, and Media Queries for responsiveness.
- **Google Fonts:** Utilizing the 'Poppins' font family.
